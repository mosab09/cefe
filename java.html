    أهم المفاهيم والأمثلة العملية لتعلم Java بسرعة

    مقدمة

    Java هي لغة برمجة قوية وموجهة للكائنات، تُستخدم لتطوير تطبيقات سطح المكتب، تطبيقات الويب، والتطبيقات المحمولة. سنتعرف هنا على أساسيات اللغة مع أمثلة عملية.

    1️⃣ أول برنامج Java

    public class Main {
      public static void main(String[] args) {
        System.out.println("مرحبا بالعالم!");
      }
    }

    البرنامج الأساسي لطباعة نص على الشاشة.

    2️⃣ المتغيرات والأنواع

    int age = 25; // عدد صحيح double height = 1.75; // رقم عشري boolean isStudent = true; // بوليان String name = "أحمد"; // نص

    3️⃣ العمليات الحسابية

    int sum = 5 + 3; int diff = 10 - 4; int mult = 6 * 2; double div = 8.0 / 2.0; int mod = 7 % 3;

    4️⃣ الشروط

    if (age >= 18) {
      System.out.println("بالغ");
    } else {
      System.out.println("قاصر");
    }

    5️⃣ الحلقات

    for (int i=0; i<5; i++) {
      System.out.println(i);
    }

    حلقة for للطباعة 0-4.

    int i = 0;
    while (i < 5) {
      System.out.println(i);
      i++;
    }

    حلقة while.

    6️⃣ الدوال (Methods)

    public static int sum(int a, int b) {
      return a + b;
    }
    System.out.println(sum(5,3)); // 8

    7️⃣ المصفوفات (Arrays)

    int[] numbers = {1,2,3,4,5}; System.out.println(numbers[0]); // 1

    8️⃣ الكائنات (Objects)

    class Person {
      String name;
      int age;
    }
    Person p = new Person();
    p.name = "أحمد";
    p.age = 25;

    9️⃣ التعليقات

    int num = sc.nextInt();

    الختام

    بهذا الموقع لديك أهم مفاهيم Java للمبتدئين مع أمثلة عملية يمكن تعديلها وتشغيلها باستخدام أي محرر Java. الخطوة التالية: تعلم البرمجة الكائنية، الوراثة، والاستثناءات Exception Handling.
