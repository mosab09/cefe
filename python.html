    أهم المفاهيم والأمثلة العملية لتعلم بايثون بسرعة

    مقدمة

    Python هي لغة برمجة سهلة وقوية تُستخدم في تطوير الويب، علوم البيانات، الذكاء الاصطناعي، وأتمتة المهام. في هذا الموقع، سنتعرف على أهم مفاهيم Python مع أمثلة عملية.

    1️⃣ المتغيرات والأنواع

    name = "أحمد" # نص age = 25 # رقم صحيح height = 1.75 # رقم عشري is_student = True # بوليان

    2️⃣ العمليات الحسابية

    sum = 5 + 3 diff = 10 - 4 mult = 6 * 2 div = 8 / 2 mod = 7 % 3 exp = 2 ** 3

    3️⃣ الشروط

    if age >= 18:
      print("بالغ")
    else:
      print("قاصر")

    4️⃣ الحلقات

    for i in range(5):
      print(i)

    حلقة for من 0 إلى 4.

    i = 0
    while i < 5:
      print(i)
      i += 1

    حلقة while.

    5️⃣ القوائم (Lists)

    fruits = ["تفاح", "موز", "برتقال"] print(fruits[0]) # تفاح fruits.append("كيوي")

    6️⃣ القواميس (Dictionaries)

    person = {"name":"أحمد", "age":25}
    print(person["name"]) # أحمد

    7️⃣ الدوال (Functions)

    def greet(name):
      return "مرحباً " + name
    print(greet("أحمد"))

    8️⃣ السلاسل النصية (Strings)

    text = "مرحبا" print(text.upper()) # MARHBA print(text.lower()) # مرحبا print(len(text)) # طول النص

    9️⃣ المجموعات (Tuples & Sets)

    point = (10, 20) # Tuple ثابت colors = {"أحمر","أخضر","أزرق"} # Set فريد

    🔟 استيراد المكتبات (Modules)

    import math print(math.sqrt(16)) # 4.0 from random import randint print(randint(1,10)) # رقم عشوائي بين 1 و 10

    الختام

    بهذا الموقع لديك أهم مفاهيم Python للمبتدئين مع أمثلة عملية يمكن تعديلها وتشغيلها. الخطوة التالية: تعلم التعامل مع الملفات، القوائم المتقدمة، البرمجة الكائنية OOP، وبيانات الويب.
