    تعلم كل تاغ HTML مع أمثلة عملية

    مقدمة

    HTML هي لغة هيكلة صفحات الويب. كل عنصر في الصفحة يُكتب باستخدام <tag>. في هذا الموقع سنستعرض أكثر من 100 تاج HTML مع الأمثلة والشروحات.

    1️⃣ الهيكل الأساسي للصفحة

    كل صفحة HTML تحتاج إلى هيكل أساسي:

    <!DOCTYPE html>
    <html lang="ar">
    <head>
    <meta charset="UTF-8">
    <title>عنوان الصفحة</title>
    </head>
    <body>
    المحتوى هنا
    </body>
    </html>

    2️⃣ العناوين والفقرات

    <h1>...</h1>

    أكبر عنوان في الصفحة.

    <h2>...</h2>

    عنوان فرعي.

    <p>...</p>

    فقرة نصية.

    <br>

    سطر جديد (Break).

    <hr>

    خط أفقي لتقسيم المحتوى.

    3️⃣ القوائم

    <ul><li>...</li></ul>

    قائمة غير مرتبة (نقاط).

    <ol><li>...</li></ol>

    قائمة مرتبة (أرقام).

    <dl><dt></dt><dd></dd></dl>

    قائمة تعريفية (تعريف ومصطلح).

    4️⃣ الروابط والصور

    <a href="URL">نص الرابط</a>

    لإنشاء رابط لموقع آخر.

    <img src="image.jpg" alt="وصف">

    لإدراج صورة.

    5️⃣ الأقسام والمنطقية

    <header>...</header>

    رأس الصفحة أو القسم.

    <footer>...</footer>

    تذييل الصفحة.

    <section>...</section>

    قسم مستقل من الصفحة.

    <article>...</article>

    محتوى مستقل مثل مقال أو منشور.

    <nav>...</nav>

    قسم خاص بالتنقل (قوائم الروابط).

    6️⃣ النصوص والفورمات

    <strong>...</strong>

    نص غامق.

    <b>...</b>

    نص غامق أيضًا (للتنسيق فقط).

    <em>...</em>

    نص مائل للتأكيد.

    <i>...</i>

    نص مائل.

    <mark>...</mark>

    نص مميز (مظلّل).

    <small>...</small>

    نص صغير.

    <del>...</del>

    نص محذوف.

    <ins>...</ins>

    نص مضاف.

    7️⃣ الجداول

    <table> <tr> <td> </td> </tr> </table>

    لإنشاء جدول.

    <th>عنوان العمود</th>

    عنوان عمود في الجدول.

    <caption>عنوان الجدول</caption>

    عنوان الجدول.

    8️⃣ النماذج (Forms)

    <form>...</form>

    لإنشاء نموذج تفاعلي.

    <input type="text">

    حقل نصي.

    <input type="password">

    حقل كلمة مرور.

    <input type="submit">

    زر إرسال للنموذج.

    <textarea>...</textarea>

    حقل نص طويل.

    <select> <option>...</option> </select>

    قائمة اختيار.

    <label>...</label>

    تسمية لحقل في النموذج.

    9️⃣ الوسائط

    <audio controls></audio>

    إدراج صوت.

    <video controls></video>

    إدراج فيديو.

    <iframe src="URL"></iframe>

    إدراج صفحة أخرى داخل الصفحة.

    10️⃣ عناصر أخرى مهمة

    <div>...</div>

    لتجميع عناصر بدون معنى محدد.

    <span>...</span>

    لتنسيق جزء صغير من النص.

    <meta>

    معلومات عن الصفحة في head.

    <link>

    ربط CSS أو أي موارد خارجية.

    <script>

    إدراج كود JavaScript.

    <style>

    كتابة CSS مباشرة داخل HTML.

    <blockquote>...</blockquote>

    اقتباس طويل.

    <q>...</q>

    اقتباس قصير.

    <abbr>...</abbr>

    اختصار مع tooltip.

    <cite>...</cite>

    مصدر أو مرجع.

    <code>...</code>

    كود برمجي.

    <pre>...</pre>

    نص مسبق التنسيق.

    <kbd>...</kbd>

    مفتاح لوحة المفاتيح.

    <samp>...</samp>

    ناتج برنامج.

    <var>...</var>

    متغير في كود برمجي.

    <sub>...</sub>

    نص أسفل السطر.

    <sup>...</sup>

    نص فوق السطر.

    الختام

    بهذا الموقع لديك أكثر من 100 تاج HTML مع شرح وأمثلة عملية. يمكنك تعديل الأكواد مباشرة لتتعلم التجربة العملية بسرعة.

    الخطوة التالية: تعلم CSS لتجميل صفحاتك ثم JavaScript لجعلها تفاعلية!
