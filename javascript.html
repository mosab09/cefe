    أهم المفاهيم والأمثلة العملية لتعلم JS بسرعة

    مقدمة

    JavaScript هي لغة برمجة تُستخدم لجعل صفحات الويب تفاعلية. يمكنك التحكم بالمحتوى، الأحداث، الرسوم، وإضافة الديناميكية للمواقع.

    1️⃣ إضافة JavaScript

    <script>alert("مرحباً!");</script>

    كتابة JS داخل ملف HTML.

    <script src="script.js"></script>

    ربط ملف JS خارجي.

    2️⃣ المتغيرات

    let name = "أحمد";

    تعريف متغير قابل للتغيير.

    const pi = 3.14;

    متغير ثابت لا يمكن تغييره.

    var age = 25;

    طريقة قديمة لتعريف متغير.

    3️⃣ الأنواع (Data Types)

    let number = 10; // رقم let name = "أحمد"; // نص let isTrue = true; // بوليان let x; // undefined

    4️⃣ العمليات الحسابية

    let sum = 5 + 3; let diff = 10 - 4; let mult = 6 * 2; let div = 8 / 2; let mod = 7 % 3;

    5️⃣ الشروط (Conditions)

    if (age >= 18) {
      console.log("بالغ");
    } else {
      console.log("قاصر");
    }

    6️⃣ الحلقات (Loops)

    for (let i=0; i<5; i++) { console.log(i); }

    حلقة for للطباعة 0-4.

    let i = 0; while(i<5){ console.log(i); i++; }

    حلقة while.

    7️⃣ الدوال (Functions)

    function greet(name) {
      return "مرحباً " + name;
    }
    console.log(greet("أحمد"));

    دالة ترجع رسالة ترحيب.

    8️⃣ الأحداث (Events)

    <button onclick="alert('تم الضغط!')">اضغط هنا</button>

    تنفيذ JS عند الضغط على زر.

    9️⃣ DOM – التحكم في الصفحة

    document.getElementById("demo").innerHTML = "تم التغيير!";

    تغيير محتوى عنصر بالصفحة.

    document.querySelector(".box").style.backgroundColor = "red";

    تغيير لون خلفية عنصر.

    10️⃣ المصفوفات (Arrays)

    let fruits = ["تفاح","موز","برتقال"]; console.log(fruits[0]); // تفاح

    11️⃣ الكائنات (Objects)

    let person = {
      name: "أحمد",
      age: 25
    };
    console.log(person.name);

    12️⃣ الارتباط مع HTML

    <p id="demo">النص الأصلي</p>
    <button onclick="document.getElementById('demo').innerHTML='تم التغيير!'">تغيير النص</button>

    الختام

    بهذا الموقع لديك أهم مفاهيم JavaScript للمبتدئين مع أمثلة عملية يمكن تعديلها وتجربتها مباشرة على المتصفح. الخطوة التالية: تعلم DOM Events، التخزين المحلي (LocalStorage)، وAJAX لتصبح مطور ويب متكامل.
