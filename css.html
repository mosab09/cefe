    تعلم خصائص CSS مع أمثلة عملية

    مقدمة

    CSS هي لغة تصميم صفحات الويب. تستخدم لتنسيق النصوص، الألوان، الخلفيات، التخطيطات، والتحكم في شكل كل عناصر الصفحة.

    1️⃣ طريقة إضافة CSS

    <style> h1 { color: red; } </style>

    كتابة CSS داخل ملف HTML في وسم <head>

    <link rel="stylesheet" href="style.css">

    ربط ملف CSS خارجي.

    <h1 style="color: blue;">عنوان</h1>

    كتابة CSS مباشرة داخل وسم HTML.

    2️⃣ الألوان والخلفيات

    color: red;

    لون النص.

    background-color: yellow;

    لون خلفية العنصر.

    opacity: 0.5;

    شفافية العنصر.

    3️⃣ النصوص والخطوط

    font-size: 20px;

    حجم الخط.

    font-family: Arial, sans-serif;

    نوع الخط.

    text-align: center;

    محاذاة النص.

    font-weight: bold;

    سمك الخط.

    4️⃣ الحدود والتنسيق

    border: 2px solid black;

    حد حول العنصر.

    border-radius: 10px;

    تدوير الزوايا.

    padding: 10px;

    المسافة الداخلية داخل العنصر.

    margin: 15px;

    المسافة الخارجية حول العنصر.

    5️⃣ العرض والارتفاع

    width: 200px;

    عرض العنصر.

    height: 100px;

    ارتفاع العنصر.

    max-width: 500px;

    أقصى عرض للعنصر.

    min-height: 50px;

    أقل ارتفاع للعنصر.

    6️⃣ التخطيط (Layout)

    display: block;

    العنصر يظهر ككتلة.

    display: inline;

    العنصر يظهر بجانب العناصر الأخرى.

    display: flex;

    تخطيط مرن للعناصر الداخلية.

    position: relative;

    موضع العنصر نسبي لمكانه الطبيعي.

    position: absolute;

    موضع العنصر مطلق بالنسبة للأب.

    float: left;

    تحريك العنصر لليسار.

    clear: both;

    إيقاف العناصر العائمة من حوله.

    7️⃣ تأثيرات النصوص

    text-decoration: underline;

    خط تحت النص.

    text-transform: uppercase;

    تحويل النص إلى أحرف كبيرة.

    letter-spacing: 2px;

    تباعد بين الحروف.

    line-height: 1.5;

    ارتفاع السطر.

    8️⃣ الخلفيات والتدرجات

    background-image: url('image.jpg');

    إضافة صورة خلفية.

    background-repeat: no-repeat;

    عدم تكرار الصورة.

    background-size: cover;

    تكبير الصورة لتغطية العنصر.

    background: linear-gradient(to right, red, yellow);

    تدرج لوني.

    9️⃣ التفاعل والانتقالات

    transition: all 0.3s ease;

    تأثير انتقالي سلس عند تغير الخصائص.

    :hover { color: red; }

    تغيير اللون عند مرور الماوس.

    الختام

    بهذا الموقع لديك أهم خصائص CSS مع أمثلة عملية يمكن تعديلها مباشرة. يمكنك الآن التعمق أكثر وتجربة CSS مع HTML لتصميم مواقعك الخاصة.
